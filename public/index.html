Mis Proyectos

proyectos

Este proyecto representa la implementación de un formulario web completamente funcional, diseñado con HTML y CSS puros. Su propósito principal es recopilar información del usuario de manera clara, ordenada y visualmente atractiva. El diseño fue pensado para garantizar una experiencia amigable en todo tipo de dispositivos, gracias a su estructura responsive que se adapta perfectamente a pantallas pequeñas como móviles, tablets o computadoras de escritorio.El formulario incluye campos esenciales como nombre, correo electrónico, mensaje o cualquier otro dato que se requiera según el contexto. También incorpora buenas prácticas de maquetación como el uso de etiquetas semánticas, organización con metodología BEM, y animaciones suaves que mejoran la interacción del usuario.

Este proyecto es una aplicacion web que permite a los usuarios crear un perfil y compartir sus datos con otros usuarios. Este proyecto fue desarrollado utilizando HTML, CSS y JavaScript. El objetivo del proyecto fue crear una interfaz de usuario atractiva y fácil de usar para los usuarios.
